# Quizgame-Laender-C-
C# Schoolproject, 


This project is a **console-based quiz game written in C**.  
The program tests the user's knowledge about **countries and geography** through a series of questions.

The player answers questions in the terminal and receives feedback based on the correctness of the answers.

---

# Features

- Console-based quiz game
- Multiple geography questions
- User input handling
- Score calculation
- Simple game loop

The game allows the player to answer questions about **countries, capitals, or geographic information**.

---

# Gameplay

Example flow of the game:


Welcome to the Country Quiz!

Question 1:
What is the capital of Germany?

a) Berlin
b) Vienna
c) Bern

Your answer: a

Correct!


At the end of the game, the player receives the **final score**.
